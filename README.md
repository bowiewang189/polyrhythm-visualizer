# Polyrhythm Web App

A full Vite + React project for the polyrhythm playground.

## Included

- real stop fix for live playback
- hit flash effect when a ball reaches a beat
- full React/Vite app
- customizable patterns
- custom or random plan sequence
- editable sequence durations
- per-pattern sound panel
- uploaded sample support
- WAV export using OfflineAudioContext
